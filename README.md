## Dependencies

LabMelee requires the following dependencies:

- [LabSDL2](https://github.com/neilpate/LabSDL2)
- [LabBASSMOD](https://github.com/neilpate/LabBASSMOD2)

### Setup

These dependencies need to be cloned into a directory parallel to the main LabMelee source directory. For example:

```
ParentFolder/
├── LabMelee/
├── LabSDL2/
└── LabBASSMOD2/
```
